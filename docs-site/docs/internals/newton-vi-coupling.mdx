# Newton Viscous-Inviscid Coupling

This document describes the implementation of XFOIL's Newton viscous-inviscid coupling algorithm in RustFoil.

## Overview

The viscous-inviscid coupling is the core of XFOIL's boundary layer solver. It couples the boundary layer equations to the inviscid flow solution through the **mass defect** variable:

```text
mass_defect = Ue * delta_star
```

Where `Ue` is the edge velocity and `delta_star` is the displacement thickness.

## Key Components

### 1. VTI Signs

The **VTI** (Viscous-Tangent-Inward) sign array handles the direction convention for upper and lower surfaces:

- Upper surface: `VTI = +1`
- Lower surface: `VTI = -1`

This affects the VM matrix construction:

```rust
// XFOIL formula
let u_m_j = -vti[i] * vti[j] * dij[(i, j)];
```

For single-surface mode, all VTI values are `+1.0`.

### 2. Forced Changes (DUE, DDS)

When there's a mismatch between the current edge velocity and what UESET would compute from the mass defect, **forced changes** are added to the Newton residuals:

```rust
// Mismatch: current Ue vs Ue computed from mass defect
let due = ue_current[i] - ue_from_mass[i];
let dds = -delta_star / ue * due;  // Induced delta_star change

// Add to residuals
rhs[i][k] += vs1_delta[k] * dds1 + vs2_delta[k] * dds2
           + vs1_ue[k] * due1 + vs2_ue[k] * due2;
```

### 3. VM Matrix

The VM matrix couples mass defect changes at all stations to the boundary layer equations:

```text
VM[i,j,k] = VS2[k,delta_star] * D[2,Mj] + VS2[k,Ue] * U[2,Mj]
          + VS1[k,delta_star] * D[1,Mj] + VS1[k,Ue] * U[1,Mj]
```

Where:
- `D_Mj = d(delta_star) / d(mass_j) = 1 / Ue` for the local station
- `U_Mj = d(Ue) / d(mass_j) = -VTI_i * VTI_j * DIJ_{i,j}`

### 4. BLSOLV Algorithm

The block-tridiagonal solver follows XFOIL's algorithm with a key difference: **row 3 (mass) is normalized by VM diagonal**, not VA:

```rust
// Forward sweep: normalize row 3 by VM(3,IV,IV)
let pivot = vm_mod[iv][iv][2];
let pivot_inv = 1.0 / pivot;

// Back substitution: use mass change to update upstream
for iv in (2..n).rev() {
    let mass_delta = solution[iv][2];
    for kv in (1..iv).rev() {
        for k in 0..3 {
            solution[kv][k] -= vm_mod[kv][iv][k] * mass_delta;
        }
    }
}
```

### 5. UPDATE Routine

The XFOIL-style update computes new edge velocities **before** applying any updates:

1. Compute new `Ue` from proposed mass changes
2. Compute global relaxation factor from ALL changes
3. Apply ALL updates with common relaxation

```rust
// Step 1: Compute new Ue from proposed mass changes
let proposed_masses: Vec<f64> = stations.iter().zip(deltas.iter())
    .map(|(s, d)| s.mass_defect + d[2]).collect();
let new_ue = compute_ue_from_mass_vec(&proposed_masses, ue_inviscid, dij, vti);

// Step 2: Compute global relaxation
let mut rlx = 1.0;
for each station:
    let dn_theta = delta[1] / theta;
    let dn_mass = delta[2] / mass_defect;
    let dn_ue = (new_ue - u) / 0.25;
    // Limit relaxation based on max change

// Step 3: Apply all updates
for each station:
    theta += rlx * delta[1];
    mass_defect += rlx * delta[2];
    u += rlx * (new_ue - u);
    delta_star = mass_defect / u;
```
